**Welcome to my mod!**
-

This mod allows you to create beautiful glass blocks that glow! The glass blocks can act as a source of light for your building!

I decided to make this because I hated using torches and oil lamps / lanterns etc. to decorate my home.

To craft the glow glass in survival mode, you need to use 2 glass blocks and 1 torch next to each other, in the pattern glass block, torch, glass block.

**Features**
- Maximum illumination (equal to torch)
- Survival friendly (easy to craft)
- Compatible with Minecraft 26.1.2
